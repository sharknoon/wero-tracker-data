# Data for the Wero Adoption Tracker

This repository contains all the data for the [Wero Adoption Tracker](https://werotracker.eu) ([GitHub](https://github.com/sharknoon/wero-tracker)).

## How to contribute

This project heavily relies on up-to-date informations about the current state of the Wero Adoption of individual banks.

If you want to help this project by reviewing, changeing or adding data, please fork this repository and make a Pull Request.
